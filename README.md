# 🎓 Edmey – LMS Website

Edmey is a **Learning Management System (LMS)** built using modern web technologies.  
It allows **educators to create and manage courses** and **students to enroll and track progress**.

---

## 🚀 Tech Stack

- Frontend: HTML, CSS, JavaScript, React.js
- Styling: Tailwind CSS
- Authentication: Clerk
- Version Control: Git & GitHub

---

## 🔐 Authentication

- Secure Login & Signup using Clerk
- Role-based dashboard after login
  - Student Dashboard
  - Educator Dashboard

---

## 👨‍🏫 Educator Dashboard Features

- 📊 Dashboard
  - Total Earnings
  - Total Students Enrolled
  - Total Courses

- ➕ Add Course  
  Educators can create and list new courses.

- 📚 My Courses  
  View all listed courses and delete courses if needed.

- 👨‍🎓 Student Enrolled  
  See how many students enrolled in each course.

---

## 👨‍🎓 Student Dashboard Features

- 📘 My Enrollments  
  View enrolled courses and completion status.

- 🎥 Watch Course Videos  
  Students can watch videos directly from dashboard.

---

## 🏠 Home Page

- Displays all courses listed by educators.
- Students can browse and enroll in courses easily.
